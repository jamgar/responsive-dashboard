# Responsive Dashboard

This is a responsive dashboard. It is part of my 100DaysOfCode challenge. 

### Version
1.0.0

## Usage
Download or clone the project.

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```
